# cylindricalbox-maker

3Dプリント用のカスタマイズ可能な円柱型の箱を作成するためのOpenSCADコードを生成するWebベースのツールです。

シンプルなインターフェースを備えており、箱の寸法を入力する4つのフィールドと、対応するOpenSCADコードが即座に生成されるテキストエリアが用意されています。

## デモ

実際に試してみる: **[https://code4fukui.github.io/cylindricalbox-maker/](https://code4fukui.github.io/cylindricalbox-maker/)**

## 機能

- 箱の寸法をカスタマイズ: 直径、高さ、側面の厚さ、底面の厚さ。
- 指定したパラメータに基づいてOpenSCADコードを即座に生成。
- デスクトップとモバイルブラウザの両方に対応した、シンプルで機能的なレイアウト。

## 使い方

1. **ツールを開く:** [デモページ](https://code4fukui.github.io/cylindricalbox-maker/)にアクセスします。
2. **寸法を設定:** 箱の直径、高さ、側面の厚さ、底面の厚さをミリメートル単位で入力します。コードはページ読み込み時に生成され、ボタンをクリックすると更新されます。
3. **コードをコピー:** テキストエリアから生成されたコード全体を選択してコピーします。
4. **モデルをレンダリング:** コピーしたコードをOpenSCADエディタに貼り付けます。これによりカスタムの円柱型の箱がレンダリングされ、3Dプリント用にエクスポートできるようになります。

## 必要条件

生成されたコードを使用するには、ソリッド3D CADオブジェクトを作成するためのフリーソフトウェアである[OpenSCAD](https://openscad.org/)が必要です。
